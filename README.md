## STATUS: Em desenvolvimento

Este projeto tem por objetivo criar uma **calculadora Web** inspirada em modelos Casio simples.

O projeto protagonizará a utilização eficiente de **funções de memória**, 
presentes na maior parte das calculadoras.
Será possível armazenar, somar, subtrair e limpar valores da memória da calculadora
Além disso, programa contará com os caracteres de 0 a 9, as quatro operações básicas.

A princípio, será desenvolvido com **HTML**, **CSS** e **JavaScript** Vanilla.

Eventuais alterações e upgrades estarão nesta seção do repositório.
